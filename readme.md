# 🎮 游戏集合

这是一个的创新互动游戏合集，通过摄像头捕捉用户的手部动作和身体姿态，实现多种沉浸式游戏体验。所有游戏均使用纯 Web 技术开发，无需安装任何插件。

## 📋 项目概览

本项目包含 **6 个** 独立的互动项目，每个项目都展示了 Web 图形技术与 AI 的结合应用：

| 项目                | 目录       | 描述                  | 核心技术                      |
| ------------------- | ---------- | --------------------- | ----------------------------- |
| 🍉 动作切水果       | `game1/` | 用手指切飞出的水果    | MediaPipe Hands               |
| 👻 隐身斗篷         | `game2/` | 实时人像分割隐身效果  | MediaPipe Selfie Segmentation |
| ✊✋✌️ 石头剪刀布 | `game3/` | 摄像头识别手势对战    | MediaPipe Hands               |
| 💥 手控星球爆炸     | `boom/`  | 拳头控制星球爆炸      | MediaPipe Hands               |
| 🧩 3D 手势拼图      | `game4/` | 捏合手指组装 3D 零件  | MediaPipe Hands + Three.js    |
| 🌟 太阳系运行图     | `star/`  | 3D 模拟太阳系行星运行 | Three.js                      |

## 🍉 Game 1: 动作切水果 (Motion Slice)

用你的食指像刀一样切开飞出的水果，但要小心炸弹！

### 功能特性

- ✋ **手部追踪**：实时追踪手指位置
- 🔪 **切水果**：食指模拟刀片切开水果
- 💣 **炸弹机制**：切到炸弹游戏结束
- ✨ **粒子特效**：华丽的切割特效
- 🏆 **计分系统**：实时显示得分

### 如何游玩

1. 打开 `game1/index.html`
2. 允许浏览器访问摄像头
3. 点击 "Start Game" 开始
4. 举起手，伸出食指，开始切水果！

---

## 👻 Game 2: 隐身斗篷 (Invisibility Cloak)

模拟哈利波特的隐形斗篷效果，实时将你从画面中"消失"！

### 功能特性

- 📸 **背景捕捉**：捕捉无人背景图像
- 🧍 **人像分割**：实时识别并分割人像
- 🪄 **隐身效果**：将人像替换为背景，实现隐身
- 🎨 **实时处理**：流畅的视频处理效果

### 如何游玩

1. 打开 `game2/index.html`
2. 允许浏览器访问摄像头
3. 点击 "Capture Background" 捕捉背景（人需离开画面）
4. 回到画面中，观看隐身效果！

---

## ✊✋✌️ Game 3: 石头剪刀布 (Rock Paper Scissors)

通过摄像头识别你的手势，与电脑对战经典的石头剪刀布游戏！

### 功能特性

- 🤲 **手势识别**：识别石头、剪刀、布三种手势
- 🤖 **AI 对战**：与电脑进行对决
- 📊 **得分统计**：记录胜负平局数
- 🎯 **实时反馈**：显示识别结果和对战结果

### 如何游玩

1. 打开 `game3/index.html`
2. 允许浏览器访问摄像头
3. 做出手势：拳头=石头，张开五指=布，伸出两指=剪刀
4. 查看对战结果！

---

## 💥 Game 4: 手控星球爆炸 (Hand-Controlled Planet)

用握拳手势让星球爆炸，用手掌移动星球位置！

### 功能特性

- 👊 **手势控制**：握拳触发爆炸，张开手掌移动
- 🌍 **星球渲染**：精美的星球视觉效果
- 💥 **爆炸动画**：震撼的爆炸粒子系统
- 🎮 **实时交互**：手部位置实时控制星球移动

### 如何游玩

1. 打开 `boom/index.html`
2. 允许浏览器访问摄像头
3. 张开手掌移动星球
4. 握拳触发星球爆炸！

---

## 🧩 Game 5: 3D 手势拼图 (3D Gesture Puzzle)

体验像钢铁侠一样的未来科技！通过手势控制光标，抓取并组装 3D 机械零件。

### 功能特性

- 🤏 **手势交互**：移动手掌控制光标，捏合手指(Pinch)抓取物体
- 🔄 **3D 操控**：在 3D 空间中移动和旋转零件
- 🛠️ **虚拟组装**：将散落的零件组装成完整的机械底盘
- 🎮 **沉浸体验**：结合 MediaPipe 和 Three.js 的 AR 体验

### 如何游玩

1. 打开 `game4/index.html`
2. 允许浏览器访问摄像头
3. **移动手掌**：控制屏幕上的光标移动
4. **捏合拇指和食指**：抓取零件，移动到目标位置后松开
5. 完成所有零件的组装！

## 🌟 Bonus: 太阳系运行图 (Solar System Simulation)

一个基于 WebGL 的 3D 太阳系模拟器，支持真实天文数据模拟。

### 功能特性

- 🪐 **双模式**：支持"模拟演示"和"真实位置"两种模式
- 🕹️ **完全控制**：可调节公转/自转速度、时间流速、光照强度等
- 📅 **真实数据**：根据 J2000.0 天文数据计算行星实时位置
- 🌌 **精美视觉**：高清贴图、星云背景和光影效果

### 如何游玩

1. 打开 `star/index.html`
2. 选择 **"模拟演示"** 模式调整速度观察行星运行
3. 或选择 **"真实位置"** 查看当前的太阳系状态
4. 使用鼠标拖拽旋转视角，滚轮缩放

## 🚀 快速开始

### 环境要求

- 现代浏览器（推荐 **Chrome** 或 **Edge**）
- 可用的网络摄像头
- 本地服务器（可选，推荐用于开发）

### 运行方式

#### 方法 1：直接打开（推荐用于快速体验）

直接双击任意游戏目录下的 `index.html` 文件即可在浏览器中运行。

#### 方法 2：本地服务器（推荐用于开发）

```bash
# 使用 Python 3 启动本地服务器
cd game1  # 或 game2, game3, boom
python3 -m http.server

# 然后在浏览器打开
# http://localhost:8000
```

## 🛠️ 技术栈

### 核心技术

- **HTML5 Canvas** - 游戏渲染和图形绘制
- **Vanilla JavaScript** - 游戏逻辑实现
- **CSS3** - 界面样式和动画

### MediaPipe 库

- **MediaPipe Hands** - 手部关键点检测和手势识别
- **MediaPipe Selfie Segmentation** - 人像分割和背景替换

### 媒体 API

- **WebRTC getUserMedia** - 摄像头访问
- **Canvas API** - 视频处理和渲染

## ⚠️ 注意事项

1. **摄像头权限**：首次运行需要授予浏览器摄像头访问权限
2. **光线环境**：良好的光线环境可以提高手势识别准确度
3. **浏览器兼容性**：建议使用最新版本的 Chrome 或 Edge 浏览器
4. **性能**：部分游戏对设备性能有一定要求，建议使用性能较好的设备

## 🎯 开发说明

所有游戏均采用模块化设计，主要包含三个文件：

- `index.html` - 页面结构和 MediaPipe 库引入
- `style.css` - 界面样式和视觉效果
- `script.js` - 游戏逻辑和 MediaPipe 集成

如需修改或扩展功能，可直接编辑对应文件。
