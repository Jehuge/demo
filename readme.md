# Motion Slice (动作切水果)

这是一个基于浏览器的动作控制游戏，使用 MediaPipe 进行手部追踪。

## 游戏简介

Motion Slice 是一款互动网页游戏，玩家可以通过网络摄像头使用食指在屏幕上切水果。游戏包含水果切片、炸弹躲避和计分系统。

## 功能特性

- **手部追踪**：使用 MediaPipe Hands 实时追踪手部动作。
- **切水果**：用食指模拟刀片切开飞出的水果。
- **炸弹机制**：切到炸弹会导致游戏结束。
- **粒子特效**：切开水果时会有绚丽的粒子效果。
- **计分系统**：记录并显示当前得分。

## 如何开始

1. 确保您的设备连接了网络摄像头。
2. 使用现代浏览器（推荐 Chrome）打开 `game1/index.html` 文件。
3. 允许浏览器访问摄像头。
4. 点击 "Start Game" 按钮开始游戏。
5. 举起手，伸出食指，开始切水果！

## 技术栈

- **HTML5 Canvas**: 用于游戏渲染。
- **MediaPipe Hands**: 用于手势识别和追踪。
- **Vanilla JavaScript**: 游戏逻辑实现。
